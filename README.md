# SCSS-in-Blazor
Getting started with SCSS in Blazor

### Prerequirements
- Blazor project
- Node js

### Set up SCSS
- Create package.json: npm init -y
- Ingore node_modules
- Install: node-sass
- Add scss compiler script
- Create PreBuildEvent
- Ingnore css files excluding cssfiles in wwwroot folder
- Create scss file
